# Reviews.io Endpoints for Cockpit CMS

## Installation
Install Cockpit CMS addon by extracting to the addons folder (/addons/ReviewsIO)

### Install Dependencies

```
$ cd /addons/ReviewsIO
$ composer install
```

### Add Rezdy API Key to Config

```
reviews-io:
    store: YOUR_STORE_ID,
    api_key: YOUR_API_KEY
```
